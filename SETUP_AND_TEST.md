# 설정 및 테스트 가이드

## 🚨 중요: CORS 문제 해결

브라우저에서 직접 Notion API를 호출할 수 없으므로, 다음 중 하나의 방법을 선택해야 합니다.

---

## 방법 1: 로컬 서버 실행 (권장)

### 1단계: Node.js 설치 확인

```bash
node --version
npm --version
```

Node.js가 설치되어 있지 않다면 [nodejs.org](https://nodejs.org)에서 설치하세요.

### 2단계: 의존성 설치

```bash
cd chat-notion-scheduler
npm install
```

### 3단계: 서버 실행

```bash
npm start
```

서버가 실행되면 다음 메시지가 표시됩니다:
```
✅ 서버가 http://localhost:3000 에서 실행 중입니다.
📱 브라우저에서 http://localhost:3000 를 열어주세요.
```

### 4단계: 브라우저에서 테스트

브라우저에서 `http://localhost:3000`을 열고 테스트를 진행하세요.

---

## 방법 2: Vercel에 배포

### 1단계: Vercel CLI 설치

```bash
npm install -g vercel
```

### 2단계: 프로젝트 배포

```bash
cd chat-notion-scheduler
vercel
```

### 3단계: 배포된 URL 접속

Vercel이 제공하는 URL로 접속하여 테스트하세요.

---

## 📋 테스트 체크리스트

### 사전 준비

- [ ] Notion 데이터베이스에 다음 속성 추가 (선택사항):
  - `이메일` (Email 타입)
  - `알림상태` (Select 타입, 옵션: "대기중", "발송완료")

### 기본 기능 테스트

1. **텍스트 입력 테스트**
   - [ ] "내일 오후 3시 팀 회의" 입력
   - [ ] 전송 버튼 클릭
   - [ ] Notion 데이터베이스에 페이지 생성 확인
   - [ ] 날짜: 내일 날짜
   - [ ] 시간: 오후 3시 (15:00)
   - [ ] 제목: 팀 회의

2. **음성 입력 테스트** (Chrome/Edge만 지원)
   - [ ] 🎤 버튼 클릭
   - [ ] "다음 주 월요일 저녁 7시 저녁 약속" 말하기
   - [ ] 입력 필드에 텍스트 자동 입력 확인
   - [ ] 전송 버튼 클릭
   - [ ] Notion 페이지 생성 확인

3. **다양한 날짜 형식 테스트**
   - [ ] "오늘 오후 2시 미팅"
   - [ ] "2026년 7월 15일 오전 10시 프로젝트 발표"
   - [ ] "다음 주 회의"
   - [ ] "모레 점심 약속"

### Outlook 이메일 알림 테스트 (선택사항)

**전제 조건:**
- Notion 데이터베이스에 `이메일`, `알림상태` 속성 추가
- Notion 자동화 2개 생성 (즉시 알림, D-1 알림)

1. **즉시 알림 테스트**
   - [ ] 일정 등록
   - [ ] 5분 이내에 이메일 수신 확인
   - [ ] 이메일 제목: "[일정 알림] {일정 제목}"
   - [ ] 이메일 내용: 날짜, 시간, 설명 포함

2. **D-1 알림 테스트**
   - [ ] 내일 일정 등록
   - [ ] 오늘 오후 6시에 이메일 수신 확인
   - [ ] 이메일 제목: "[내일 일정 알림] {일정 제목}"

## 🐛 문제 해결

### "Notion 연동 실패: Failed to fetch" 오류

**원인:** 로컬 서버가 실행되지 않았거나 CORS 문제

**해결:**
1. 로컬 서버가 실행 중인지 확인: `http://localhost:3000`
2. 서버 재시작: `npm start`
3. 브라우저 콘솔에서 에러 메시지 확인

### "이메일 is not a property that exists" 오류

**원인:** Notion 데이터베이스에 이메일 속성이 없음

**해결:**
1. 이메일 기능을 사용하지 않으려면 설정에서 "Outlook 알림" 체크 해제
2. 이메일 기능을 사용하려면 Notion 데이터베이스에 속성 추가:
   - 속성 이름: `이메일`
   - 타입: Email
   - 속성 이름: `알림상태`
   - 타입: Select (옵션: "대기중", "발송완료")

### 음성 입력 버튼이 보이지 않음

**원인:** 브라우저가 Web Speech API를 지원하지 않음

**해결:**
- Chrome 또는 Edge 브라우저 사용
- Firefox, Safari는 제한적 지원

### 서버 실행 시 "npm: command not found"

**원인:** Node.js가 설치되지 않음

**해결:**
1. [nodejs.org](https://nodejs.org)에서 Node.js 설치
2. 터미널 재시작
3. `node --version` 명령으로 설치 확인
