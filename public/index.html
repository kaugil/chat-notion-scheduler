설정 방법

📝 Notion 설정

  1. Notion Integration 생성:
    • Notion Integrations 페이지로 이동
    • "New integration" 클릭
    • 이름 입력 후 "Submit" 클릭
    • 생성된 "Internal Integration Token" 복사
  2. 데이터베이스 생성 및 연결:
    • Notion에서 새 페이지 생성
    • "/database" 입력 후 "Table - Inline" 선택
    • 다음 속성 추가:
      • 제목 (Title)
      • 날짜 (Date)
      • 시간 (Text)
      • 설명 (Text)
    • 페이지 우측 상단 "..." → "Add connections" → 생성한 Integration 선택
    • 페이지 URL에서 Database ID 복사 (예: notion.so/xxxxxxxx...)

📧 Outlook 설정

  1. MCP 서버 설정 필요:
    • Outlook 연동은 MCP(Model Context Protocol) 서버를 통해 작동합니다
    • 서버 관리자에게 Microsoft 365 앱 등록 및 MCP 서버 설정을 요청하세요
    • 자세한 내용은 /Users/ygkwon/Documents/IBM Bob/MCP/outlook-server/README.md 참조
  2. 이메일 주소 입력:
    • 알림을 받을 회사 이메일 주소를 입력하세요
    • D-1 알림을 활성화하면 일정 하루 전 오전 9시에 자동으로 이메일이 발송됩니다
